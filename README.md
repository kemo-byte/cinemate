# Cinemate

Cinemate is a modern movie browsing application built with React. It allows users to explore movies by categories such as "Now Playing," "Popular," "Top Rated," and "Upcoming," or search for specific movies. The app is powered by the TMDB API and features a responsive design with dark mode support.

## Features

- **Movie Categories**: Browse movies by categories like Now Playing, Popular, Top Rated, and Upcoming.
- **Search Functionality**: Search for movies by title.
- **Movie Details**: View detailed information about individual movies.
- **Dark Mode**: Toggle between light and dark themes.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used

- **React**: Frontend library for building the user interface.
- **React Router**: For client-side routing.
- **Tailwind CSS**: For styling and responsive design.
- **TMDB API**: To fetch movie data.
- **Custom Hooks**: For reusable data-fetching logic.
